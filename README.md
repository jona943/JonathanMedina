## Estructura del Proyecto

El sitio está construido con una estructura clásica de archivos para facilitar el mantenimiento y la escalabilidad:

-   `index.html`: La página principal o de aterrizaje. Contiene la estructura semántica del contenido principal.
-   `portafolio.html`: Página dedicada a mostrar los proyectos más relevantes.
-   `sobre_mi.html`: Sección que detalla mi viaje profesional y mis pilares de trabajo.
-   `css/styles.css`: Hoja de estilos principal. Define la apariencia visual, el diseño responsive y las animaciones del sitio.
-   `js/main.js`: Archivo de JavaScript para la interactividad del cliente, como el menú de navegación móvil (hamburguesa).
-   `img/`: Directorio que contiene las imágenes de la interfaz y fotografías.
-   `icone/`: Almacena los iconos tecnológicos utilizados en la sección de habilidades.

## Características Técnicas

-   **Diseño Responsive:** La interfaz se adapta a diferentes tamaños de pantalla (móvil, tablet, escritorio) utilizando media queries en CSS.
-   **Navegación Interactiva:** El menú de navegación incluye un botón "hamburguesa" funcional para dispositivos móviles, controlado con JavaScript.
-   **Estructura Semántica:** Se utiliza HTML semántico para mejorar la accesibilidad y el SEO.
-   **Estilos Modulares:** El CSS está organizado para ser legible y mantenible.

## Cómo Visualizar el Proyecto

Para ver el sitio en tu máquina local, sigue estos pasos:

1.  **Clona el repositorio:**
    ```bash
    git clone https://github.com/jona943/JonathanMedina.git
    ```
2.  **Navega al directorio del proyecto:**
    ```bash
    cd JonathanMedina
    ```
3.  **Abre el archivo `index.html` en tu navegador web preferido.**
